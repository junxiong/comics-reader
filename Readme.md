# Ray's Comic Reader

A cross platform online/offline comic reader

## Supported platforms
  * Windows
  * Mac osx
  * Linux
  * Android (to be done)
  * IOS (to be done)

## Main feature
  * Online comics searching
  * Offline comic shelf
  * Offline reading history
  * Offline comics cache
